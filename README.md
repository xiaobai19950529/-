# -

一，安装Nodejs

  1. Github上获取源码
  
     sudo git clone https://github.com/nodejs/node.git
     
  2. 修改目录权限
  
     sudo chmod -R 755 node
     
  3. 使用./configure创建编译文件
  
     sudo ./configure
     sudo make
     sudo make install
     
  4. 查看版本号
  
     node --version
